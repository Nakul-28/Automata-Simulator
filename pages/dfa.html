DFA Simulator Guide

1) Build The State Diagram

  • Choose Add State, then click on the canvas to create states.
  • Use Select to click a state and edit name, start, and accept properties in State Properties.
  • Choose Add Transition, drag from source state to target state, then set symbols in Transition Symbol.

2) Canvas Tools

  • Select: choose states/transitions and drag to move states or transition curves.
  • Add State: inserts new nodes at click position.
  • Add Transition: freehand drag between states to create curved transitions.
  • Use Undo/Redo to revert or restore edits quickly.

3) Export And Import JSON

  • Export JSON saves the current automaton (nodes, edges, symbols, positions).
  • Import JSON opens the template modal. Pick a valid .json file to load a diagram.
  • After import, the JSON editor opens so you can modify and Apply JSON immediately.

4) Transition Table Feature

  • Open Transition Table to view rows as states and columns as input symbols.
  • Each DFA cell must contain at most one target state label. Leave blank to remove transition.
  • Add State/Add Symbol inserts new row/column. Delete buttons remove rows/columns.
  • Apply To Diagram writes table values into the canvas diagram. Reset From Diagram discards table edits.
