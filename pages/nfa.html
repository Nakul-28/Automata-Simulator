NFA Simulator Guide

1) Build The State Diagram

  • Choose Add State, then click on the canvas to place states.
  • Use Select to update state name, start, and accept properties in State Properties.
  • Choose Add Transition and drag source to target, then enter symbols (comma-separated).
  • Use ε in Transition Symbol for epsilon transitions.

2) Canvas Tools

  • Select: choose states/transitions and drag to move states or curve control points.
  • Add State: creates a new state at click location.
  • Add Transition: creates curved transitions by drag gesture.
  • Undo/Redo helps recover from mistakes while constructing the automaton.

3) Export And Import JSON

  • Export JSON saves full automaton structure and geometry.
  • Import JSON validates and loads a diagram from file.
  • Imported content also opens in the JSON editor where you can update and Apply JSON.

4) Transition Table Feature

  • Rows represent states, columns represent symbols (including ε when used).
  • Each cell may contain multiple target states separated by commas (example: q1, q2).
  • Leave a cell blank to delete that transition for the selected symbol.
  • Add State/Add Symbol creates table structure; Apply To Diagram syncs table data back to canvas.
